Projects

research

Enabling Multi-priority Coding in LED-based Optical Camera Communications

We propose a novel hierarchical coding scheme named as OnionCode to support dynamic range of channel capacity in one-to-many OCC scenario.
Applications of OIS beyond Image Stabilization

We explore the method to control optical image stabilization (OIS) module of phone camera without additional hardware and develop applications including depth maps and super-resolution images.
Nonlinearity of Spatial Frequency in Light and Its Applications

We model the nonlinear interaction between Color Filter Array and screens, and build applications inlcluding the novel optical encryption method for QR codes on top of our model.
Magic Finger Input System

We propose a training-free text input system called MagicInput that support multiple languages using acoustic-based 1D finger tracking technology. An artificial dataset (called TrackMNIST) simulated from MNISTseries datasets is utilized to satisfy the various users' writing characteristics.
User Identification based on Leakage Current

We developed a behavior-irrelevant user identification system applicable to laptops with a metal casing based on leakage current.
Handwriting Recognition System based on Vibration Signal

We presented a novel handwriting recognition system based on vibration signals detected by the built-in accelerometer of smart phones.
Enhancing QRCode Security by Fingerprinting Screens

In ScreenID, PWM frequency of screens is exploited as the unique screen fingerprint and used to enhance the security of a QR code by identifying its authenticity.
Utilizing Acoustic-based Imaging for Issuing Contact-Free Silent Speech Commands

A silent speech interface based on acoustic signals.
Near-filed communication via EM side-channel

A novel near-filed communication that utilizes EMI signals emitted from CPUs to transmit data and magnetometers on mobile devices to receive data.
MagThief

MagThief outlines a novel monitoring system to enable the theft of sensitive app usage information without the need for user permissions on the mobile devices.
